# Anxiety Management App #

Este projeto trata-se de um aplicativo mobile health gamificado para auxílio no controle da ansiedade de jovens do ensino-médio.

# Objetivo do projeto

Auxiliar jovens no controle da ansiedade por meio de tecnicas distrativas. Servindo como apaziguador dos sintomas e indentificador de causas que podem ajudar em um possivel diagnostico.

# Pacotes usados

[FlutterBloc](https://pub.dev/packages/flutter_bloc) e [Equatable](https://pub.dev/packages/equatable) pra gerencia de estado.

FirebaseAuth para autenticação.

GraphQl com Hasura para banco de dados.
